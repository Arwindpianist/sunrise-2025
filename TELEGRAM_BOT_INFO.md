## Technical Notes

### Webhook Configuration
The bot should be configured to receive webhooks from your Sunrise 2025 application for:
- Event notifications
- Contact updates
- Message delivery status
- User interactions

### Message Templates
The bot should support rich text formatting including:
- Bold, italic, underline
- Links and buttons
- Inline keyboards for actions
- Media attachments (images, documents)

### Rate Limiting
Ensure the bot respects Telegram's rate limits:
- 30 messages per second
- 20 messages per minute per chat
- Proper error handling for rate limit exceeded

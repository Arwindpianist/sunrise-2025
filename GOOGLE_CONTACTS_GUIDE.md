# 📱 Google Contacts Export Guide

## 🎯 Overview
This guide will help you export your contacts from Google Contacts and import them into the Sunrise app. This is the most reliable method for importing contacts.

## 📋 Prerequisites
- A Google account with contacts
- Access to a web browser
- The exported CSV file

## 🚀 Step-by-Step Instructions

### Step 1: Access Google Contacts
1. **Open your web browser**
2. **Go to**: https://contacts.google.com
3. **Sign in** with your Google account if not already signed in

### Step 2: Navigate to Export
1. **Look for the left sidebar** (hamburger menu on mobile)
2. **Click "Export"** in the sidebar menu to open export options
3. **If you don't see Export**, look for "More" or "Settings" first
4. **On mobile**, the Export option might be in the hamburger menu (☰)

### Step 3: Configure Export Settings
1. **Select contacts to export**:
   - ✅ **All contacts** (recommended)
   - ✅ **Specific groups** (if you want only certain contacts)
2. **Choose export format**:
   - ✅ **Google CSV** (recommended)
   - ❌ Avoid other formats for best compatibility

### Step 4: Download the File
1. **Click "Export"** button
2. **Wait for download** to complete
3. **Note the file location** (usually Downloads folder)
4. **File name format**: `contacts.csv` or `google_contacts.csv`

### Step 5: Import to Sunrise
1. **Go back to Sunrise app**
2. **Click "Import Contacts"**
3. **Select "Google Contacts (Recommended)"**
4. **Click "Upload File"**
5. **Choose the downloaded CSV file**
6. **Select a category** (optional)
7. **Click "Import Contacts"**

## 📱 Mobile Instructions

### Android Users
1. **Open Chrome browser**
2. **Go to**: https://contacts.google.com
3. **Follow the same steps** as desktop
4. **Download will go to Downloads folder**

### iOS Users
1. **Open Safari browser**
2. **Go to**: https://contacts.google.com
3. **Follow the same steps** as desktop
4. **Download will go to Files app**

## 🔧 Troubleshooting

### ❌ "Export" not visible
**Solution**: 
- Look for "More" or "Settings" in the sidebar first
- On mobile, check the hamburger menu (☰) for Export
- Try refreshing the page
- Make sure you're signed into the correct Google account
- The Export option appears after you have contacts in your account

### ❌ Download not starting
**Solution**:
- Check browser popup blockers
- Try a different browser
- Check your internet connection

### ❌ File format issues
**Solution**:
- Make sure you selected "Google CSV" format
- Don't use "Outlook CSV" or other formats
- Try exporting a smaller group of contacts first

### ❌ Import fails
**Solution**:
- Check that the CSV file contains email addresses
- Make sure the file isn't corrupted
- Try exporting again with different settings

## 📊 Expected Results

### ✅ Successful Import
- You'll see a success message
- Contacts will appear in your contact list
- Only contacts with email addresses will be imported

### ⚠️ Partial Import
- Some contacts may be skipped if they lack email addresses
- You'll see a warning message with details
- Check the console for specific error messages

## 🎯 Tips for Best Results

### Before Export
1. **Clean up your contacts** - remove duplicates
2. **Ensure email addresses** are present for important contacts
3. **Organize into groups** if you want selective imports

### During Import
1. **Choose appropriate category** for organization
2. **Review preview** before final import
3. **Check console** for any warnings or errors

### After Import
1. **Verify contacts** were imported correctly
2. **Add missing information** manually if needed
3. **Organize into categories** for better management

## 🔗 Alternative Methods

If Google Contacts export doesn't work:

1. **File Upload**: Export contacts from your phone's contacts app
2. **Native Picker**: Use the built-in contact picker (limited browser support)
3. **Manual Entry**: Add contacts one by one

## 📞 Support

If you continue having issues:

1. **Check browser console** for error messages
2. **Try different browsers** (Chrome, Safari, Firefox)
3. **Contact support** with specific error details
